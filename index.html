How to Use - ncexs Toolkit

Quick Run (No Download Required)

1

Open your Start Menu, type powershell, right-click it and select Run as Administrator.

2

Copy and paste the command below, then press Enter:

Set-ExecutionPolicy Bypass -Scope Process -Force; [Net.ServicePointManager]::SecurityProtocol = [Net.SecurityProtocolType]::Tls12; irm "https://raw.githubusercontent.com/ncexs/ncexs-toolkit/main/ncexs-Toolkit.ps1" | iex
3

The toolkit will launch automatically.

4

To cancel or exit any running process, press Ctrl + C.

To return and use the toolkit again, simply copy and paste the command below:

Set-ExecutionPolicy Bypass -Scope Process -Force; [Net.ServicePointManager]::SecurityProtocol = [Net.SecurityProtocolType]::Tls12; irm "https://raw.githubusercontent.com/ncexs/ncexs-toolkit/main/ncexs-Toolkit.ps1" | iex

How to Use - ncexs AutoTask

Download Script

Download the official PowerShell automation script from our GitHub repository:

Download ncexs-AutoTask.ps1

Manual Run Command

  1. Open PowerShell (Administrator recommended).
  2. Execute the script locally with this command:
    powershell -ExecutionPolicy Bypass -File ncexs-AutoTask.ps1
Automate with Task Scheduler (Recommended)
1
Open Task Scheduler

Press Win + R, type taskschd.msc, and press Enter.

2
Create a New Task

Click Create Task on the right sidebar (not Basic Task). In the General tab:

  • Name: ncexs Auto Task
  • Check: "Run whether user is logged on or not"
  • Check: "Run with highest privileges"
3
Configure Trigger (Schedule)

Go to the Triggers tab, click New. Set trigger to On a schedule, select Daily. Optionally, check "Repeat task every:", type 6 hours, and set duration to Indefinitely.

4
Configure Action

Go to the Actions tab, click New, set Action to "Start a program":

  • Program/script: powershell.exe
  • Arguments: -ExecutionPolicy Bypass -File "C:\Path\To\ncexs-AutoTask.ps1" -Silent
  • (Change folder path to where you saved the script. Omit -Silent for interactive debugging)
5
Configure Conditions

In the Conditions tab, uncheck: "Start the task only if the computer is on AC power" (this allows it to run on laptops on battery power).

6
Configure Settings & Save

In the Settings tab, check "Run task as soon as possible after a scheduled start is missed". Click OK and enter your password if prompted.

How to Use - ncexs Junk Cleaner

One-Click Run Instructions

1

Download the latest release ZIP from the GitHub Releases.

2

Extract the downloaded ZIP package.

3

Double-click the ncexs-junkcleaner.bat file to launch the script.

4

If SmartScreen blocks execution ("Windows protected your PC"), click More info → Run anyway.

5

Allow Administrator access when prompted by User Account Control (UAC).

6

Follow the on-screen prompts (type Y or N) for: Junk Cleaner, Deep Cleanup, Antivirus Quick Scan, or RAM optimization.

Key Feature Explanation
  • Change Language (Menu L):
    Switches the toolkit's interactive console interface between English and Indonesian instantly.
  • Compact OS (Menu 0):
    A powerful Windows feature that compresses system binaries. It is 100% safe, reversible, and can instantly free up 2GB to 5GB of storage space without deleting any files.
  • System Information (Menu 1):
    Displays quick and detailed specifications of your computer, including CPU, RAM, Windows version, and active drive space.
  • Enhanced Junk Cleaner (Menu 2):
    Safely removes temporary files, system log files, and browser caches (Chrome, Edge, Brave). Also cleans shader caches for AMD, NVIDIA, and Intel GPUs to fix game stuttering issues.
  • Empty Recycle Bin (Menu 3):
    Instantly empties the Windows Recycle Bin for all users and drives to free up space.
  • Open Disk Cleanup (Menu 4):
    Launches the official Windows Disk Cleanup utility (cleanmgr) for advanced system file cleanup.
  • Smart Uninstaller (Menu 5):
    Quickly search and remove installed programs using fuzzy logic search. Just type the app name or initials!
  • Network Repair (Menu 6):
    Runs a complete 5-step network reset: Release/Renew IP, Flush DNS, Reset Winsock, and Reset TCP/IP stack. Useful for fixing connection drops.
  • Power & Battery Utilities (Menu 7):
    Generates a detailed HTML battery health report, configures high performance power plans, and manages system sleep states.
  • Memory Optimizer (Menu 8):
    Instantly clears system standby memory and releases unused RAM caches, helping to speed up system responsiveness and games.
  • Defrag & Optimize Drives (Menu 9):
    Analyzes and runs TRIM command for SSDs or defragments HDDs to maintain high drive read/write speeds.
  • System Health Checker (Menu 10):
    Runs SFC (System File Checker) and DISM repair tools to find and fix corrupt Windows system files automatically.
  • Updates & Drivers Center (Menu 11):
    Fixes stuck or failed Windows Updates by resetting the update services (wuauserv, bits, cryptSvc) and renaming the download cache folder.
  • DNS Changer (Menu 12):
    Allows you to quickly change your DNS server settings to popular secure options like Google Public DNS or Cloudflare for faster, safer browsing.
  • Wi-Fi Password Recovery (Menu 13):
    Quickly retrieves and displays saved Wi-Fi profiles and passwords on your current machine.
  • Visual FX Booster (Menu 14):
    Optimizes system visual effects and animations to boost speed and performance, especially on older hardware.
  • Optimize WhatsApp (Menu 15):
    Cleans cached files, logs, and unnecessary media sent/received via WhatsApp Desktop to free up gigabytes of wasted storage.
  • Exit Toolkit (Menu 16):
    Safely closes the console window and restores your original PowerShell execution policies.
